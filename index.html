Difference between Frontend and Backend

The front end is everything related with what the user sees, and what he can interact with. It is normally associated with web development.

On the other hand, the back end is everything on the server side and that which makes the front end work accordingly. Developers are in charge of this area

Who is the end user?

The end user refers to the individual to whom the application is intended for. Developers have to take in mind and listen to the opinions and needs of the users that will most likely use their website or application.

Who is the client?

The client is normally the individual financing a project and in really rough terms he is going to dictaminate what he wants included in the project. The developer has to ask questions to his client that are related to aspects that he needs in order to program and meet its client expectations.

CDN architecture

The Content Delivery Networks are the medium or the part in Internet network in charge of, as its name indicates, delivering the content a user may request upon entering a website, downloading media or in the use of instant messaging. Physical distances matter when we are trying to request content from a server far away.

History of HTML and CSS

HTML was created in 1990 as a way to provide organized text documents, it could be formatted for different font sizes, later came implementation of images and tables. At some point Web browsers started appearing and using this html language but it was difficult to implement any kind of visual value. Then arround 1996 CSS came around separating the presentation and the content. CSS added the much needed style, color, fonts and any kind of visual elements while saving time.
